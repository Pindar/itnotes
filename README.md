Pindar.github.io
================

This is the source code of itnotes.de.

## Development

### Prerequisits

* Ruby
* Jekyll: `gem install jekyll`
* Bundler: `gem install bundler`
* `xcode-select --install`

### Run

1. bundle install
1. `bundle exec jekyll build` to build
1. `bundle exec jekyll serve` to run the blog locally
